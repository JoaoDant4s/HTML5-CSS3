Principais formatações

Negrito / destaque

Nessa frase temos um termo em negrito usando a tag (não semântica) < b >
Entretanto, nesta outra frase temos um termo em destaque usando a tag (semântica) < strong >

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag < i >
Entretanto, nesta outra frase temos um termo em destaque usando a tag < em >

Texto marcado

Podemos criar também um texto marcado usando a tag < mark >

Texto grande e pequeno

Segue um exemplo de um texto grande e de um texto pequeno usando a tag < big > (não semântica) e a tag < small >

Texto deletado

Segue um exemplo de um texto que deve ser lido mas não considerado usando a tag < del >

Texto inserido

Podemos marcar um texto como inserido ( com a tag < ins >) para dar enfâse e indicar que ele foi adicionado depois.
Existe também o texto sublinhado com a tag (não semântica) < u >

Texto sobrescrito

Para inserir coisas do tipo: 220+3
usa-se a tag < sup >

Texto subescrito

Para inserir coisas do tipo: H2O
usa-se a tag < sub >
